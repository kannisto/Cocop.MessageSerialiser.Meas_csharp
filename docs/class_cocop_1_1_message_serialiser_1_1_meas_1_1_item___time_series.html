Detailed Description

Abstract base class for time series. A common base class exists, because the XML structures of time series resemble each other very much. However, in C#, not all structures can be reused. This applies especially because of collection initialisers that require the implementation of the IEnumerable interface.

In this module, the code has been derived from TimeseriesML 1.0 â€“ XML Encoding of the Timeseries Profile of Observations and Measurements (OGC 15-042r3; please see the file "ref_and_license_ogc_tsml.txt").
