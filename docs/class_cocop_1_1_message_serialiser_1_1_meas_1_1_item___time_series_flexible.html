Detailed Description

A time series of measurement values. In this time series type, there is no requirement for constant time intervals between measurement points.

In this module, the code has been derived from TimeseriesML 1.0 â€“ XML Encoding of the Timeseries Profile of Observations and Measurements (OGC 15-042r3; please see the file "ref_and_license_ogc_tsml.txt").
